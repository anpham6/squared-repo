# squared-repo

## Install

- [Repo](https://source.android.com/docs/setup/download#installing-repo) (*Python*)

## Usage

```bash
mkdir workspace
cd workspace

repo init -u https://github.com/anpham6/squared-repo -m nightly.xml // prod.xml (patches)
repo sync

cd squared
```
